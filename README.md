# Excel-Dashboard-1
# Developed an interactive Excel dashboard to analyze business performance across multiple dimensions, including revenue, profit, expenses, payment methods, product lines, and geographical regions.
# Key Highlights
# 1. Cleaned and transformed raw transactional data using Excel functions and data preparation techniques.
# 2. Built dynamic dashboards using Pivot Tables, Pivot Charts, and slicers for interactive analysis.
# 3. Analyzed revenue, expenses, and profit across departments such as Sales, Marketing, HR, Operations, and R&D.
# 4. Evaluated product line performance and identified top revenue-generating categories.
# 5. Compared customer payment preferences across Bank Transfer, Cash, Credit Card, and PayPal transactions.
# 6. Examined regional transaction trends across Africa, Asia-Pacific, Europe, North America, and South America.
# 7. Designed visually appealing KPI cards to track total revenue, expenses, profit, and transaction metrics.

# Tools Used
# 1. Microsoft Excel
# 2. Pivot Tables & Pivot Charts
# 3. Data Cleaning & Transformation
# 4. Dashboard Design & Data Visualization
# 5. Business Performance Analysis
